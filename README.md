# Just An Ordinary Walk Home
This program is a simple text adventure game in which you choose the character's actions 
that result in different outcomes.

---

## How to Run
All you need is Pycharm or your Operating System's command prompt to run it.
You need both the main.py file and the scenes_and_functions.py in order to run the game.
You run the game from the main.py file

# How to Play
When prompted, Enter your name and press enter.
When prompted, input the requested values/responses and press ENTER to confirm your input.
